Brazier
essentials:brazier
With a <item>Brazier/$, many different things can be done. An item can be added or removed with a right click, or automation.
Adding some water with a bucket makes it safe to land on the <item>brazier/$ from a high height, and makes it extinguish you if you stand on it.
Adding lava instead makes it set things on it on fire, so it can be used as a trash can for items that won't set the area on fire.
A coal block makes it burn, letting off light, without consuming the coal.
Similarly, a glowstone block also makes it glow at light level 15.
Soul sand prevents the spawning of witches and endermen teleportation in a large area (default 32 block radius). This is useful for mob farms.
