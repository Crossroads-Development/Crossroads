The <item>Life-Spark capsule/$ creates an <item><link:essentials:witchcraft/cloning#process>embryo/$ from a <item><link:essentials:witchcraft/blood_centrifuge>centrifuged blood pouch/$, as part of the <thing><link:essentials:witchcraft/cloning>cloning process/$.
Ingredients have to be added to the <item>Life-Spark capsule/$ using either a <item><link:essentials:witchcraft/syringe#embryo_lab>syringe/$ or a dispenser. If adding items using a <item><link:essentials:witchcraft/syringe#embryo_lab>syringe/$, hold the <item><link:essentials:witchcraft/syringe#embryo_lab>syringe/$ in the main hand and the item being added in the offhand, and right click.
The <item><link:essentials:witchcraft/blood_centrifuge>centrifuged blood pouch/$ must be added first- the <item>empty blood pouch/$ will be returned.
Additional items can be added to add special <thing><link:essentials:witchcraft/cloning#traits>traits/$. Adding a potion will add the potion effect (returning the empty bottle), adding a nametag will add <thing><link:essentials:witchcraft/cloning#traits>imprinting/$, and adding a <item><link:essentials:witchcraft/souls>soul cluster/$ will add <thing><link:essentials:witchcraft/cloning#traits>respawning/$. An (uncentrifuged) <item>blood sample/$ from a player can be added to add <thing>imprinting/$, specifically targeting the player the sample is from rather than the player who spawns the mob.
To finalize the crafting, a lightning rod/<item><link:essentials:fluid/dynamo#lightning_rod>lightning rod/$ extension on top of the <item>Life-Spark capsule/$ needs to be hit by lightning- any direct strike by lightning (such as a <thing><link:essentials:beam/charge>charge beam/$) will work. This will consume all the ingredients and create one <thing><link:essentials:witchcraft/spoilage>fresh/$ <item><link:essentials:witchcraft/cloning#process>embryo/$. If the <item><link:essentials:witchcraft/blood_centrifuge>centrifuged blood pouch/$ was originally spoiled, it will add 1 <thing><link:essentials:witchcraft/cloning#degradation>degradation/$.
<thing><link:essentials/circuits>Circuits/$ read 1 if there is a <item><link:essentials:witchcraft/blood_centrifuge>centrifuged blood pouch/$ in the machine, otherwise 0.
